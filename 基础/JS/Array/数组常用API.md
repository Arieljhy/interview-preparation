## 数组 常用 API
 
 - push() pop() shift() unshift()
  
 - slice(start,end)  //不改变原数组
 - splice(start,count) //改变原数组
  
 - join()
 - reverse()
 - concat()
 - fill()
 - sort()
  
 ### 数组 遍历
 - forEach()
 - reduce()
 - map() //改变原数组
  
 ### 数组 查找
 - indexOf()
 - lastIndexOf()
 - findIndex()
 - includes()
  
 - filter()
 - some()
 - any()
 - every()
 
 ### 数组 转字 符串
 - toString()
 - toLocalString()
  
 ### 转化 为 数组
 - Array.from()
  
 ### 判断 是否为 数组
 - Array.isArray()    //true 是 ，false 不是

`收集报错`
- js执行错误: window.onerror()
- 资源加载错误: 
    window.addEventListener('error', e => {
        report({type: 'resource-error', message: e.message})
    })
- Promise 未处理异常:
    window.addEventListener('unhandlerejection', e => {
        report({type: 'promise-rejection', message: e.message})
    })

#### 比如说一个 React 组件做性能优化有哪些措施呢
***`核心原则：减少不必要的渲染 和 计算`*** 是一个持续迭代的过程

`组件级别的优化：`
- 按需导入
- 使用React.memo处理组件，避免组件的不必要的渲染

`hooks级别的优化：`
- useCallback ：处理函数，避免子组件的不必要的渲染
- useMemo： 缓存计算结果，避免不必要的重计算

`长列表优化：`
- 列表虚拟化：计算并仅渲染可见区域Item

`代码分割：`
- React.lazy() ：动态导入路由
- 按需导入组件
- <Suspense> 处理懒加载组件 的加载状态

`第三方库：`
优先选择提供 ES Module 版本的库（如 lodash-es）
